# Amore Site

Home corporativa da Amore Beneficios, direcionada a empresas e profissionais de RH que buscam ampliar os beneficios oferecidos aos colaboradores.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Node.js 22

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Build de producao

```bash
npm run build
npm run start
```

O projeto usa Next.js padrao e pode ser importado diretamente pela Vercel a partir do repositorio GitHub.
